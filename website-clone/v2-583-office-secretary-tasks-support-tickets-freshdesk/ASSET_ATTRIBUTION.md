# Asset attribution

## Freshdesk local mark

`static/assets/freshdesk-mark.svg` is an original task-replica asset authored for this repository. It uses a generic headset/support motif and is not copied from the production Freshdesk logo. It is served only from the local replica origin.

## Lucide icons

`static/vendor/lucide.min.js` is a vendored browser build of Lucide, copied from an existing repository replica to avoid a runtime CDN request.

- Project: Lucide
- Website: https://lucide.dev/
- License: ISC
- Local license: `static/vendor/LICENSE.lucide`

No runtime request is made to Lucide, Freshdesk, Freshworks, a font CDN, analytics, advertising, or any other external service.
