# Asset attribution

All runtime assets are original, repository-local files created for this
offline replica.

- `static/assets/idealist-mark.svg`: original local geometric lettermark. It is
  not copied from Idealist's trademark artwork.
- `static/styles.css`: original responsive styling using system fonts.
- `static/app.js`: original task-scoped client implementation.

No source-site image, font, script, stylesheet, CDN asset, tracker, or other
external runtime resource is bundled or requested.

Public first-party semantics were checked against:

- https://www.idealist.org/en/program-management-jobs-washington-dc
- https://www.idealist.org/en/nonprofit-job/d8cc46076acf4feaae09189314a34e24-program-manager-dumbarton-arts-education-washington

Only normal-user facts and interaction semantics were adapted. No protected
source code or downloaded source asset is included.
