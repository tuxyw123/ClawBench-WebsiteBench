Agent website reconstruction benchmark

Can an agent rebuild a website it can only explore?

WebsiteBench gives an agent an isolated offline website—not its source code. The agent must interact with it, infer its behavior, and produce a new runnable clone. This viewer makes the input website, the generated clone, and every score inspectable.
