VISUAL INTERPRETATION视觉指标解释

Direct checks and diagnostics直接检查与诊断指标

Gate 3 uses direct pixel-derived comparisons only where frozen source paint is usable and dimensions are compatible. SSIM, edge overlap, color histograms, normalized error, heatmaps, and structural summaries are evidence for review—not substitute official scores.Gate 3 仅在冻结源站画面可用且尺寸兼容时采用直接像素对比。SSIM、边缘重合、颜色直方图、归一化误差、热图与结构摘要用于审核举证，不能替代正式成绩。

Eligible可直接比较
Usable source paint and compatible geometry.源站画面可用且几何尺寸兼容。
Structural结构诊断
DOM and layout evidence retained without a direct-visual claim.保留 DOM 与布局证据，但不作直接视觉通过声明。
Unavailable不可用
Protected, expected-error, or near-uniform source states are excluded explicitly.受保护、预期错误或近乎纯色的源站状态被明确排除。

SAFETY BOUNDARY安全边界

Read-only source, local-only clone源站只读，克隆站仅限本地

  • Source observations are anonymous and GET-only; non-GET requests are blocked before transmission.源站观察为匿名且仅限 GET；非 GET 请求在传输前即被阻断。
  • Clone runtime resources are bundled and same-origin with zero external requests.克隆站运行资源均本地打包并同源，外部请求为零。
  • Identity, addresses, payment, checkout, and real order placement are disconnected.身份、地址、支付、结账与真实下单均不连接。
  • The interactive clone exists only behind an authenticated, CSRF-protected internal gateway.交互式克隆站仅可通过带登录与 CSRF 保护的内部网关访问。
