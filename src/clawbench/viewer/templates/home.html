One scored task, one fused commerce surface一个评分任务，一个融合的电商能力面

The scored journey still discovers the number-two external SSD and commits exactly one quantity-two cart action. The same Amazon runtime now also supports local registration, verification, sign-in, guest-cart migration, checkout, orders, cancellation, and password reset. These extensions are functional benchmark content, but are not silently added to the current score.评分旅程仍要求发现排名第二的外置 SSD，并准确提交一次数量为二的购物车操作。同一 Amazon 运行时现已支持本地注册、验证、登录、游客购物车迁移、结账、订单、取消与密码重置；这些扩展是可操作的 benchmark 内容，但不会被悄悄计入当前分数。
